Programming Assignment 5 -  GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications [5 points]
  • Indentation [3 points]
  • Variable names [2 points]
  • javadoc format documentation * (see NOTE) [5 points]

    • NOTE 1: Only submit java files. We will use the javadoc *.java command to create the javadoc files.
    NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.
    NOTE 3: YOUR ASSIGNMENT WILL NOT BE GRADED if you use ArrayList, Vector, LinkedList, Tree or any predefined Java API Data Structures.

2. (18 points) Source code:
  • Implementation done as specified (additional methods and classes are OK). [13 points]
  • Exception handling and error checking as specified. [5 points]

3. (65 points) Program correctness:

  • Compiles without error (no partial credit). [20 points]
  • Check for valid filename and options. [4 points]
  • Program works correctly for the following commands: [41 points]
    • I [9 points]
    • C [4 points]
    • L [9 points]
    • N [3 points]
    • S [4 points]
    • A [5 points]
    • D [5 points]
    • Q [2 points]

4. (5 points) Extra credit:

  • Menu Option R (Read mailing records from a file) [5 points]
    Note: This option may be selected more than once. If invalid filename is given, a message must be printed.
