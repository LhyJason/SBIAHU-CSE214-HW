HOMEWORK - SPRING 2023

Programming Assignment 2 -  GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications [5 points]
  • Indentation [3 points]
  • Variable names [2 points]
  • javadoc format documentation * (see NOTE) [5 points]

    • NOTE 1: Only submit java files. We will use the javadoc *.java command to create the javadoc files.
    NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.
    NOTE 3: YOUR ASSIGNMENT WILL NOT BE GRADED if you use ArrayList, Vector, LinkedList or any predefined Java API Data Structures.

2. (18 points) Source code:
  • Implementation done as specified (additional methods are OK) [10 points]
  • Throw an exception back to the calling method(s) if a precondition is violated. [8 points]
2. (60 points) Program correctness:
  • Compiles without error (no partial credit) [9 points]
  • All specified methods work correctly:
    • Insert when list is empty, insert before head, insert before tail, and insert before other nodes. [7 points]
    • Append when list is empty or if list has at least one node. [7 points]
    • Remove cursor node when cursor is head, tail, or any other node. [7 points]
    • Print current line or a given range of lines. [7 points]
    • Move forward and print the next line. Also, test for end of list and print a message, if necessary. [3 points]
    • Move backward and print the last line. Also, test for beginning of the list and print a message, if necessary. [3 points]
    • Line numbers are printed correctly during insert, append and print. [3 points]
    • Print a message if an invalid option or invalid number is entered. [3 points]
    • Print the help menu. [3 points]
    • Print a message if user enters invalid line range. [3 points]
    • Options are NOT case sensitive (lower case or upper case are the same). [3 points]
    • If text is modified or a new line is inserted, confirm before terminating program. [2 points]
3. (5 points) Program efficiency: The efficiency of all methods should not exceed O(n).

Note: To print a list, cursorLineNo must be called only once and its value be assigned to a local counter. Then, after each line is printed, the counter should be incremented by one. This avoids multiple calls to cursorLineNo for printing a given range of lines.
