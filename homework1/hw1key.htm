Programming Assignment 1 -  GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications [5 points]
  • Indentation [3 points]
  • Variable names [2 points]
  • Javadoc Style documentation *(see NOTE). [5 points]

NOTE 1: Only submit java files. We will use the javadoc *.java command to create the javadoc files.
NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.
NOTE 3: YOUR ASSIGNMENT WILL NOT BE GRADED if you use ArrayList, Vector, LinkedList or any predefined Java API Data Structures.

2. (18 points) Source code:

  • Implementation done as specified (additional methods are OK) [8 points]
  • Throw an exception back to the calling method(s) if a precondition is violated. [10 points]

3. (60 points) Program correctness:

  • Compiles without error (no partial credit) [14 points]
  • The methods clone, equals, add, subtract, multiplyByScalar, and dotProduct work correctly. [7 points each]
    Note: The methods clone, getElement, setElement, getSize must all be used in your
    implementation; otheriwse up to 10 points will be deducted.
  • Exception handling. [4 points]

4. (5 points) Program efficiency: all operations are O(n)
