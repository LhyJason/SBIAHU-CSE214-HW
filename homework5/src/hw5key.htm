HOMEWORK 5 - GRADING KEY

1. (17 points) Documentation and Coding Style
  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications. [5 points]
  • Indentation. [3 points]
  • Variable Names. [2 points]
  • Javadoc Style for Documentation. [5 points]

NOTE 1: You only need to submit your *.java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

2. (28 points Points) Source Code
  • Implementation done as specified (additional methods are okay). [12 points]
  • Correct use of Hashtable methods in the Java API for each of the menu options [4 points each = 16 points]
    • put()
    • get()
    • remove()
3. (55 Points) Program Correctness
  • Program compiles without any errors. [15 points]
  • Program searches for "network.obj" on startup and creates a hash table if the file exists.[4 points]
  • Display the content of hash table if "network.obj" exists. [4 points]
  • Program automatically saves to "network.obj" upon completion.[4 points]
  • Menu options work correctly.[18 points]
    • Properly prompts for user profile and adds to table.[4 points]
    • Properly prompts for user name and removes from table.[4 points]
    • User profiles can be successfully updated.[4 points]
    • Properly checks to see if two users are networked.[6 points]
  • Error checking and special cases: [2 points each = 10 points]
    • Checking for insertion of invalid user Profile.
    • Checking for insertion of duplicate entry.
    • Appropriate error when Profile is not found in table.
    • Case-insensitive menus and searches.
    • Use of final variables where indicated.
4. (5 Points) Extra Credit
  • Implement Optional Extra Credit. [5 points]
